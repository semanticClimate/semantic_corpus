# A Semantic Corpus-Based Literature Review of Climate Change and Plant Growth

## Project Overview

This project demonstrates the SemanticClimate Corpus Review workflow using Europe PMC to retrieve, organize, and review scientific literature related to Climate Change and Plant Growth.

## Workflow

1. Configure literature search query
2. Retrieve publications from Europe PMC
3. Download XML articles
4. Generate review datasets
5. Perform query refinement
6. Compare literature retrieval results

## Technologies Used

- Google Colab
- Python
- SemanticClimate Corpus Review Notebook
- Europe PMC
- Pandas

## Generated Outputs

- Review Table (CSV)
- Review Table (JSON)
- Review Table (Markdown)
- Workflow Diagram
- Literature Review Dataset
- Query Comparison
- Four-Week Progress Report
